## 🎯 حالة المشروع
[![Build Android APK](https://github.com/@abdjlqahr7111/SocialApp-Rebuilt/actions/workflows/build_android.yml/badge.svg)](abdjlqahr7111)

## 📱 كيفية تشغيل البناء
1. اذهب إلى تبويب **Actions** (أعلى الشاشة)
2. اضغط على **Build Android APK** من القائمة
3. اضغط على زر **Run workflow** الأزرق
4. اختر `Release` ثم اضغط **Run workflow** الأخضر

## 📥 تحميل التطبيق
- اذهب إلى **Actions** > آخر عملية ناجحة > **Artifacts**
- حمّل الملف المناسب لجهازك:
  - `arm64-v8a` للأجهزة الحديثة
  - `armeabi-v7a` للأجهزة الأقدم

## 🛠️ التقنيات المستخدمة
- Qt6 + QML للواجهة
- C++ للمعالجة
- SQLite لقاعدة البيانات
- GitHub Actions للبناء السحابي
